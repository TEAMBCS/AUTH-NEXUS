# Security Policy

## Supported Versions

The following versions of Auth Nexus currently receive security updates.

| Version | Supported |
|--------|-----------|
| 2.0 | ✅ Supported |
| <2.0 | ❌ Not Supported |
| 1.0 | Back Dated version|

---

## Reporting a Vulnerability

If you discover a security vulnerability in Auth Nexus, please report it responsibly.

Do NOT create a public GitHub issue for security vulnerabilities.

Instead, report it privately through one of the following methods:

• GitHub Security Advisory  
• Direct contact with the maintainer  

Provide the following information in your report:

- Description of the vulnerability
- Steps to reproduce the issue
- Possible impact
- Suggested fix (if available)

We will review the report and respond as quickly as possible.

---

## Responsible Disclosure

We encourage responsible disclosure of security issues.

Please allow time for the issue to be investigated and resolved before publicly disclosing the vulnerability.

---

## Security Updates

Security patches will be released through official GitHub releases.

Users are encouraged to always use the latest version of Auth Nexus.
